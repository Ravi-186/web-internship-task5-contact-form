# Contact Form with HTML Validation

This project demonstrates a simple contact form built using HTML5 and CSS with built-in client-side validation.

## Features
- HTML5 form validation (required, pattern, maxlength)
- Email and phone number validation
- Accessible labels
- Clean and responsive UI
- No JavaScript used

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Download or clone the repository
2. Open index.html in a browser
3. Test form validation
